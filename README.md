# 개인정보 처리방침 (Privacy Policy)

**Stock Principal Calculator**(이하 "앱")은 사용자의 개인정보를 중요하게 생각하며 관련 법령을 준수합니다. 본 개인정보 처리방침은 앱 이용 시 수집되는 개인정보와 그 처리 방식에 대해 설명합니다.

## 1. 수집하는 개인정보 항목

### 1) 사용자가 직접 입력하는 정보
- 없음  
  *(본 앱은 사용자가 개인 정보를 직접 입력받지 않습니다.)*

### 2) 서비스 이용 과정에서 자동으로 수집되는 정보
- 기기 정보(모델명, 운영체제 버전 등)
- 서비스 이용 기록 및 로그 데이터
- 광고 식별자(ADID, IDFA 등)

### 3) 외부 서비스(SDK)를 통해 수집되는 정보
앱은 서비스 제공을 위해 아래 외부 서비스(SDK)를 사용할 수 있습니다.

- Google AdMob (광고 제공)
- Firebase Analytics (사용자 행동 분석)

## 2. 개인정보 수집 방법
- 사용자가 앱을 이용하는 과정에서 자동 수집  
- 외부 SDK를 통한 자동 수집

- 법령에서 별도 보관을 요구하는 경우 해당 기간 동안 보관합니다.
